## **Installation**

- Clone repository and install dependencies:

```bash
$ git clone https://github.com/matheusmichels/netflix-clone.git
$ cd netflix-clone
$ yarn global add react-native-cli
$ yarn
```

- Run application

```bash
$ react-native run-android
$ react-native run-ios
```
